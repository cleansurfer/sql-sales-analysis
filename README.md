# MySQL Sales Database Analysis

This project demonstrates database design, data insertion, and SQL analysis
using MySQL. It includes:

 Database schema  
 Sample dataset  
 Analytical SQL queries  
 Revenue analysis  
 Customer segmentation  
 Product insights  

## Files
- schema.sql → Creates the database and tables
- data.sql → Inserts sample data
- queries.sql → Contains analytical SQL queries

## Topics Covered
- JOIN, GROUP BY, ORDER BY
- Aggregation functions
- Database design
- Basic analytics

## How to Run
1. Install MySQL
2. Run: `mysql -u root -p < schema.sql`
3. Run: `mysql -u root -p < data.sql`
4. Run queries in `queries.sql`
