# MySQL Sales Database Analysis

---> Overview: This project is a structured relational database. 
It also reflects practical experience with real-world database modeling.
It demonstrates core SQL skills  such as:
 database design / data insertion / SQL analysis / Database schema design / Sample dataset / Analytical SQL queries / Revenue analysis / Customer segmentation / Product insights / table creation / primary & froreign keys / joins / aggregation queries

---> files:
schema.sql → Creates the database and tables
data.sql → Inserts sample data
queries.sql → Contains analytical SQL queries

---> topics covered :
JOIN, GROUP BY, ORDER BY
Aggregation functions
Database design
Basic analytics

---> how to run:
  1. Install MySQL
  2. Run: `mysql -u root -p < schema.sql`
  3. Run: `mysql -u root -p < data.sql`
  4. Run queries in `queries.sql`
